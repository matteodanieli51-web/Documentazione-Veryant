# Getting Started

The setup of isCOBOL IDE requires the following steps:

1. [Download and install the Java Development Kit (JDK)](/pages/is-cobol-IDE/Getting-Started/download-JDK)
2. [Download and install isCOBOL Evolve](/pages/is-cobol-IDE/Getting-Started/download-isCOBOL-Evolve)
3. [Activate the License](/pages/is-cobol-IDE/Getting-Started/activate)

In order to activate your isCOBOL Evolve products, you will need the e-mail you received from Veryant containing your license key. Contact your Veryant representative for details.

**Note** - isCOBOL IDE setups are available only for 64 bit platforms.
