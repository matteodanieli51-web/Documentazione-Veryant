# Working with Projects

## Creating a new Project

In order to use isCOBOL IDE to create and maintain COBOL programs, one or more projects must be created. A Project is a container for programs that share the same settings and resources.
To create a new Project in a workspace, right click in the [isCOBOL Explorer]() area and choose New */ isCOBOL Project* from the pop-up menu. A wizard will ask for project name and location, compiler options and additional jar libraries. Most of these settings can be changed later, after the project has been created, by clicking on *Project* menu and selecting *Properties* from the list.
