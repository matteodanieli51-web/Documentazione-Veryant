# System Requirements

## isCOBOL SDK System Requirements

### Windows

- Windows 7 SP1 or later
- RAM: 128 MB
- Disk space: 256 MB
- JDK 8, 11, 17 or 21

**Note**: As of January 14, 2020 Microsoft stopped supporting Windows 7 and therefore it is no longer an officially supported platform. Users may still continue to use isCOBOL on Windows 7 at their own risk, but support will only be provided for Microsoft Windows releases Windows 8 or later.

### Mac OS X

- Intel-based Mac running Mac OS X 10.7.3 (Lion) or later
- RAM: 128 MB
- Disk space: 256 MB
- Administrator privileges for installation
- 64-bit browser[^1]
- JDK 8, 11, 17 or 21

### Linux

- kernel version 2.6 or later
- glibc 2.3.4 or later
- RAM: 128 MB
- Disk space: 256 MB
- JDK 8, 11, 17 or 21

### Sun Solaris (SPARC - 64 bit)

- SunOS 5.10 or later
- RAM: 128 MB
- Disk space: 256 MB
- JDK 8, 11, 17 or 21

### OpenServer

- OpenServer 10
- RAM: 128 MB
- Disk space: 256 MB
- JDK 8, 11, 17 or 21

## isCOBOL IDE System Requirements

### Windows

- Windows 7 SP1 or later
- RAM: 512 MB;
- Disk space: 600 MB
- JDK 11, 17 or 21

**Note**: As of January 14, 2020 Microsoft stopped supporting Windows 7 and therefore it is no longer an officially supported platform. Users may still continue to use isCOBOL on Windows 7 at their own risk, but support will only be provided for Microsoft Windows releases Windows 8 or later.

### Mac OS X (64 bit)

- Intel-based Mac running Mac OS X 10.7.3 (Lion) or later.
- RAM: 512 MB;
- Disk space: 600 MB
- Administrator privileges for installation
- 64-bit browser[^1]
- JDK 11, 17 or 21

### Linux (64 bit)

- kernel version 2.6 or later
- GTK 3.0 or later
- RAM: 512 MB;
- Disk space: 600 MB
- JDK 11, 17 or 21

## JEE Application Server requirements 

- Tomcat 8.0 or later
- JBOSS EAP 6.4 or later
- WebLogic 12.1 or later
- WebSphere 8.5 or later
- WildFly 10 or later
  
[^1]: A 64-bit browser (Safari, for example) is required to run Oracle Java on Mac.
