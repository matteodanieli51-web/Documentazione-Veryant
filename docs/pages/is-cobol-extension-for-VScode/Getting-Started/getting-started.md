# Getting Started

The setup of the development environment requires the following steps:

1. [Download and install the Java Development Kit (JDK)](/pages/is-cobol-extension-for-VScode/Getting-Started/download-JDK)
2. [Download and install the isCOBOL Evolve SDK](/pages/is-cobol-extension-for-VScode/Getting-Started/download-isCOBOL-Evolve)
3. [Download and install Visual Studio Code](/pages/is-cobol-extension-for-VScode/Getting-Started/download-VSCode)
4. [Download and install the Veryant isCOBOL extension](/pages/is-cobol-extension-for-VScode/Getting-Started/download-Veryant-isCOBOL-extension)
5. [Activate the License](/pages/is-cobol-extension-for-VScode/Getting-Started/activate-license)

In order to activate your isCOBOL Evolve products, you will need the e-mail you received from Veryant containing your license key. Contact your Veryant representative for details.
