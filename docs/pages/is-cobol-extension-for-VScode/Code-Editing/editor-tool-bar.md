## Editor tool-bar

The editor tool-bar provides some shortcuts:

| Button | Action |
| --- | --- |
| Button | Action |
|  | Compile all the source files in the project. |
|  | Run the program specified by [Veryant > Main: Program](). If the above setting is not present, then run the current program. |
|  | Debug the program specified by [Veryant > Main: Program](). If the above setting is not present, then debug the current program. |
|  | Duplicate the current editor tab in a split-pane, so you can see different portions of the same source file at the same time. |
